# Настройка Telegram уведомлений

## Шаг 1: Создание Telegram бота

1. Открой Telegram и найди [@BotFather](https://t.me/botfather)
2. Отправь команду `/newbot`
3. Введи имя бота (например: "Flower Shop Notifications")
4. Введи username бота (должен заканчиваться на `bot`, например: `flower_shop_notifications_bot`)
5. **Скопируй токен** - это будет `TELEGRAM_BOT_TOKEN`

Пример токена: `1234567890:ABCdefGHIjklMNOpqrsTUVwxyz`

## Шаг 2: Получение Chat ID

### Вариант 1: Через @userinfobot (проще)
1. Найди бота [@userinfobot](https://t.me/userinfobot)
2. Отправь ему любое сообщение
3. Он ответит с твоим Chat ID (число, например: `123456789`)

### Вариант 2: Через @getidsbot
1. Найди бота [@getidsbot](https://t.me/getidsbot)
2. Отправь ему любое сообщение
3. Он покажет твой Chat ID

### Вариант 3: Через API (для групп)
Если нужно отправлять в группу:
1. Добавь бота в группу
2. Отправь сообщение в группу
3. Используй скрипт `test_telegram.py` для получения chat_id группы

## Шаг 3: Добавление в .env

Создай или открой файл `backend/.env` и добавь:

```env
TELEGRAM_BOT_TOKEN=1234567890:ABCdefGHIjklMNOpqrsTUVwxyz
TELEGRAM_CHAT_ID=123456789
```

## Шаг 4: Тестирование

Запусти тестовый скрипт:
```bash
python test_telegram.py
```

Если всё настроено правильно, ты получишь тестовое сообщение в Telegram!

## Примечания

- Бот должен быть запущен (не заблокирован)
- Если отправляешь в группу, бот должен быть добавлен в группу и иметь права на отправку сообщений
- Chat ID для личных сообщений - это твой личный ID
- Chat ID для групп - это ID группы (обычно отрицательное число)
