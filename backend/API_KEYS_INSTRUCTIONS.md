# Инструкция по получению API ключей для загрузки изображений

## Проблема
Изображения загружаются, но не соответствуют названиям цветов. Нужно использовать API с ключом для точного поиска.

## Варианты решения

### Вариант 1: Pexels API (РЕКОМЕНДУЕТСЯ)
**Преимущества:**
- Бесплатный доступ
- 200 запросов в час (достаточно для проекта)
- Хорошее качество изображений
- Простая регистрация

**Как получить ключ:**
1. Перейди на https://www.pexels.com/api/
2. Нажми "Get Started" или "Sign Up"
3. Зарегистрируйся (можно через Google/GitHub)
4. Создай новое приложение
5. Скопируй API ключ

**Как использовать:**
```powershell
# Установи ключ
set PEXELS_API_KEY=твой_ключ_от_pexels

# Запусти скрипт
python load_images_with_pexels_api.py
```

---

### Вариант 2: Unsplash API
**Преимущества:**
- Бесплатный доступ
- 50 запросов в час
- Огромная база изображений

**Как получить ключ:**
1. Перейди на https://unsplash.com/developers
2. Нажми "Register as a developer"
3. Зарегистрируйся или войди
4. Создай новое приложение
5. Скопируй Access Key

**Как использовать:**
```powershell
# Установи ключ
set UNSPLASH_ACCESS_KEY=твой_ключ_от_unsplash

# Запусти скрипт
python load_images_with_unsplash_new_key.py
```

---

## Какой вариант выбрать?

**Рекомендую Pexels**, потому что:
- Больше запросов в час (200 vs 50)
- Проще регистрация
- Хорошее качество изображений цветов

---

## После получения ключа

1. Установи ключ в переменную окружения
2. Запусти соответствующий скрипт
3. Проверь несколько изображений в браузере
4. Если изображения правильные - готово!
5. Если нет - можно уточнить поисковые запросы в скрипте

---

## Примечание

Оба скрипта используют **точные поисковые запросы** на английском языке:
- "white roses bouquet" для "Белые розы"
- "red freesia flowers" для "Красные фрезии"
- и т.д.

Это должно обеспечить правильное соответствие изображений названиям цветов.
