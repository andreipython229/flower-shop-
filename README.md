# Flower Shop - Лендинг по продаже цветов

Лендинг для продажи цветов на Django REST Framework (backend) + React (frontend).

## Структура проекта

```
flower-shop/
├── backend/          # Django + DRF (API)
│   ├── manage.py
│   ├── flowers/      # Приложение для цветов
│   └── ...
├── frontend/         # React (интерфейс)
│   ├── src/
│   ├── public/
│   └── ...
└── README.md
```

## Функционал

- Каталог цветов
- Корзина покупок
- Оформление заказа
- Контактная информация

## Технологии

- Backend: Django REST Framework
- Frontend: React
- База данных: SQLite (для разработки)
